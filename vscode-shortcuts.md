# VS Code Keyboard Shortcuts for Windows

```
Shortcut                    Description
--------------------------  -----------------------------------------------
Ctrl+Shift+P                Show Command Palette
Ctrl+P                       Quick Open, Go to File
Ctrl+Shift+N                 New Window/Instance
Ctrl+W                       Close Window
Ctrl+,                       User Settings
Ctrl+K Ctrl+S                Keyboard Shortcuts
Ctrl+Shift+X                 Extensions
F1                          Show Command Palette
Ctrl+K Ctrl+O                Open Folder
Ctrl+Shift+O                 Go to Symbol
Ctrl+T                       Show All Symbols
Ctrl+Shift+M                 Show Problems Panel
F8                          Go to Next Error or Warning
Shift+F8                    Go to Previous Error or Warning
Ctrl+Shift+Tab              Open Previous Recently Used Editor
Ctrl+Tab                    Open Next Recently Used Editor
Ctrl+K P                    Copy Path of Active File
Ctrl+K R                    Reveal Active File in File Explorer
Ctrl+K O                    Open Active File in New Window
Ctrl+Shift+T                Reopen Closed Editor
Ctrl+K Ctrl+W               Close All Editors
Ctrl+K Ctrl+Shift+T         Reopen Closed Editor in New Window
Ctrl+1                      Focus into 1st Editor Group
Ctrl+2                      Focus into 2nd Editor Group
Ctrl+3                      Focus into 3rd Editor Group
Ctrl+K Ctrl+Left            Move Active Editor Group Left
Ctrl+K Ctrl+Right           Move Active Editor Group Right
Ctrl+Shift+PageUp           Move Editor Left
Ctrl+Shift+PageDown         Move Editor Right
Ctrl+\                      Split Editor
Ctrl+K Ctrl+\               Split Editor Orthogonal
Ctrl+Shift+\                Jump to Matching Bracket
Ctrl+Shift+]                Expand Selection
Ctrl+Shift+[                Shrink Selection
Ctrl+Space                  Trigger Suggest
Ctrl+Shift+Space            Trigger Parameter Hints
Tab                         Emmet Expand Abbreviation
Ctrl+K Ctrl+X               Trim Trailing Whitespace
Ctrl+K Ctrl+F               Format Document
Ctrl+K Ctrl+Shift+F         Format Selection
Ctrl+Shift+I                Format Document
Alt+Shift+F                 Format Document
Ctrl+K Ctrl+C               Add Line Comment
Ctrl+K Ctrl+U               Remove Line Comment
Ctrl+/                      Toggle Line Comment
Ctrl+Shift+/                Toggle Block Comment
Alt+Shift+Down              Copy Line Down
Alt+Shift+Up                Copy Line Up
Ctrl+D                      Add Selection to Next Find Match
Ctrl+K Ctrl+D               Move Last Selection to Next Find Match
Ctrl+F                      Find
Ctrl+H                      Replace
Ctrl+Shift+F                Find in Files
Ctrl+Shift+H                Replace in Files
F3                         Find Next
Shift+F3                   Find Previous
Ctrl+G                      Go to Line
Ctrl+S                      Save
Ctrl+Shift+S                Save As
Ctrl+K S                    Save All
```
