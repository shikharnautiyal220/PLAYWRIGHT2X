# Playwright2X

Welcome to **Playwright2X**, a comprehensive tutorial series designed to take you from JavaScript fundamentals to mastering Playwright for web automation testing. This repository provides step-by-step examples, code snippets, and exercises to build your skills progressively.

## Overview

Playwright2X is structured as a learning path, starting with basic JavaScript concepts and gradually introducing Playwright's powerful features for end-to-end testing. Each chapter includes practical code examples and explanations.

## Chapters

### Chapter 01: Basic
- `01_HelloJs.js` - Introduction to JavaScript

- `03_VerifySetup.js` - Setup verification

## Prerequisites

Before getting started, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (version 14 or higher)
- [Git](https://git-scm.com/) for cloning the repository

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shikharnautiyal220/Playwright2X.git
   cd Playwright2X
   ```

2. Install Playwright (if needed for future chapters):
   ```bash
   npm install -g playwright
   npx playwright install
   ```

## How to Run

Each JavaScript file can be executed using Node.js:

1. Navigate to the desired chapter folder:
   ```bash
   cd Chapter_01_Basic
   ```

2. Run a specific file:
   ```bash
   node 01_HelloJs.js
   ```

For Playwright-specific scripts (in future chapters), use:
```bash
npx playwright test
```
